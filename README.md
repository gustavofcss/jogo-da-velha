
## **1. Descrição Geral**

O programa simula uma partida de Jogo da Velha entre dois jogadores (X e O) utilizando o terminal.
O tabuleiro é representado por uma matriz 3×3 e é atualizado conforme a interação do usuário.

O jogo encerra automaticamente quando ocorre:

* Identificação de um vencedor
* Preenchimento completo do tabuleiro (empate)

---

## **O que o jogo faz**

* Exibe um tabuleiro 3×3
* Permite que dois jogadores joguem por turno
* Impede jogadas inválidas
* Detecta vitórias nas linhas, colunas e diagonais
* Identifica empates
* Finaliza a partida automaticamente quando o jogo termina

Tudo isso de forma simples, clara e totalmente interativa pelo console.

---

## **Como jogar**

1. Execute o arquivo no terminal:

   ```sh
   python jogoDaVelha.py
   ```

2. Informe a **linha** e a **coluna** onde deseja jogar (valores de 0 a 2).

3. O tabuleiro será atualizado a cada jogada.

4. O jogo termina quando alguém vence ou quando todas as casas são preenchidas.
